# Ejecutar en Desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener Nest CLI instalado

```
npm i -g @nest-j/cli
```

4. Levantar la base de datos

```
docker-compose up -d
```

##Stack Usado
*MongoDB
*Nest
